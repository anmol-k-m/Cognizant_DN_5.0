To check if a number is prime, we need to determine if it has any divisors other than 1 and itself.

Here's a breakdown of the logic and the Python code:

1.  **Definition of a Prime Number:**
    *   A prime number is a natural number greater than 1.
    *   It has no positive divisors other than 1 and itself.

2.  **Basic Algorithm:**
    *   If the number `n` is less than or equal to 1, it's not prime.
    *   If `n` is 2, it's prime (the only even prime).
    *   If `n` is greater than 2 and is even, it's not prime.
    *   For odd numbers `n` greater than 2, we check for divisibility by odd numbers starting from 3 up to the square root of `n`.
        *   We only need to check up to the square root because if `n` has a divisor `d` greater than `sqrt(n)`, then it must also have a divisor `n/d` which is less than `sqrt(n)`. So, if we haven't found a divisor up to `sqrt(n)`, we won't find one beyond it.

```python
import math

def is_prime(number):
    """
    Checks if a given number is a prime number.

    A prime number is a natural number greater than 1 that has no positive divisors
    other than 1 and itself.

    Args:
        number (int): The integer to check for primality.

    Returns:
        bool: True if the number is prime, False otherwise.
    """

    # 1. Numbers less than or equal to 1 are not prime by definition.
    if number <= 1:
        return False

    # 2. 2 is the only even prime number.
    if number == 2:
        return True

    # 3. All other even numbers are not prime.
    if number % 2 == 0:
        return False

    # 4. For odd numbers, we check for divisors starting from 3 up to the
    #    square root of the number, incrementing by 2 (checking only odd numbers).
    #    We use math.isqrt() for integer square root, which is efficient.
    #    The loop runs up to `limit` (inclusive), so range needs `limit + 1`.
    limit = math.isqrt(number) 
    for i in range(3, limit + 1, 2):
        if number % i == 0:
            # If we find any divisor, it's not prime
            return False

    # 5. If no divisors were found, the number is prime.
    return True

# --- Test Cases ---
print(f"Is 0 prime? {is_prime(0)}")       # Expected: False
print(f"Is 1 prime? {is_prime(1)}")       # Expected: False
print(f"Is 2 prime? {is_prime(2)}")       # Expected: True

print(f"Is 17 prime? {is_prime(17)}")     # Expected: True
print(f"Is 25 prime? {is_prime(25)}")     # Expected: False (divisible by 5)
print(f"Is 29 prime? {is_prime(29)}")     # Expected: True
print(f"Is 97 prime? {is_prime(97)}")     # Expected: True

print(f"Is 100 prime? {is_prime(100)}")   # Expected: False
print(f"Is 101 prime? {is_prime(101)}")   # Expected: True
print(f"Is 127 prime? {is_prime(127)}")   # Expected: True
```

### Explanation:

1.  **`import math`**: We need the `math` module for `math.isqrt()`, which calculates the integer square root.
2.  **`number <= 1`**: Prime numbers must be greater than 1. So, any number 1 or less is immediately classified as not prime.
3.  **`number == 2`**: 2 is the smallest prime number and the only even prime number. We handle it separately for efficiency.
4.  **`number % 2 == 0`**: If the number is greater than 2 and is even (i.e., divisible by 2), it cannot be prime. We can quickly rule out all even numbers here.
5.  **`limit = math.isqrt(number)`**: This is a crucial optimization. We only need to check for divisors up to the square root of `number`.
    *   `math.isqrt(number)` computes the integer square root. For example, `isqrt(25)` is 5, `isqrt(26)` is 5.
6.  **`for i in range(3, limit + 1, 2)`**:
    *   We start checking from `i = 3` because we've already handled 2.
    *   We go up to `limit + 1` because `range()` is exclusive of the end value. If `limit` is 5, we want to check 3 and 5, so the range needs to go up to 6.
    *   `step` of `2`: We only check odd numbers (3, 5, 7, ...) as potential divisors. We don't need to check even numbers because if a number `n` had an even divisor `d`, then `n` would also be even, which we've already handled in step 3.
7.  **`if number % i == 0`**: If `number` is perfectly divisible by `i` (meaning `i` is a factor), then `number` is not prime, and we can immediately return `False`.
8.  **`return True`**: If the loop completes without finding any divisors, it means the number has no divisors other than 1 and itself (and we've already ruled out 1), so it must be prime.

This approach is efficient enough for checking the primality of numbers up to several millions or billions. For extremely large numbers, more advanced algorithms (like Miller-Rabin primality test) are used.
